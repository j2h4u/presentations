# Presentations

Static GitHub Pages site for public presentation decks.

- Index: <https://j2h4u.github.io/presentations/>
- 2026-07-01 deck: <https://j2h4u.github.io/presentations/20260701/>

## Structure

- `index.html` is the public index page.
- `YYYYMMDD/index.html` contains a standalone presentation deck.

## Analytics

Pages include GoatCounter analytics.

- Dashboard: <https://j2h4u.goatcounter.com/>
- Tracking endpoint: `https://j2h4u.goatcounter.com/count`

When adding a new HTML page, include the GoatCounter snippet before `</body>`:

```html
<script data-goatcounter="https://j2h4u.goatcounter.com/count"
        async src="//gc.zgo.at/count.js"></script>
```
